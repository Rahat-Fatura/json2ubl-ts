## 3. Publish Adımı

Tüm kapılar yeşilse:

```bash
# 1. Son temiz build
npm run build

# 2. npm login (eğer yapılmadıysa)
npm login

# 3. Publish (public access, ilk 2.x sürümü)
npm publish --access public

# Alternatif: tag ile (örn. latest tag'e push etmeden önce "next" ile test)
# npm publish --tag next --access public
```

**Güvenlik:** Publish sırasında `.env`, `audit/`, `.claude/`, `src/` gibi dizinler `files` whitelist dışında olduğu için paket içine girmez. Yine de `npm pack --dry-run` ile son doğrulama önerilir.
